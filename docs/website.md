# Сайт

Публичный сайт находится в корне проекта и состоит из HTML-страниц, общего `styles.css`, общего `script.js` и ассетов в `assets`.

## Основные страницы

- `index.html` — главная страница, афиша, услуги верхнего уровня, мастера, контакты.
- `services.html` — услуги и модальная форма заявки.
- `about.html` — история таверны.
- `dnd-simple.html` — объяснение D&D простым языком.
- `miniature-painting.html` — покраска миниатюр.
- `gallery.html` — атмосфера и воспоминания.
- `hatter-diary.html` — дневник Шляпника.
- `rating.html` — публичный рейтинг игроков.
- `privacy-policy.html` — политика обработки персональных данных.
- `masters/*.html` — страницы мастеров.

## Динамические данные

`script.js` получает:

- афишу через `GET /api/games`;
- рейтинг через `GET /api/rating`;
- настройки контактов из `data/site-settings.js`.

Если API недоступен, сайт показывает ошибку, а не демонстрационные карточки.

## Формы

Формы записи на игру и заказа услуги отправляются в backend:

- `POST /api/game-signups`
- `POST /api/service-requests`

Факт согласия на обработку персональных данных передаётся вместе с заявкой и дополнительно проверяется backend.

## Локальное открытие

Статические страницы можно открыть через `file://`, но динамическая афиша, рейтинг и отправка заявок требуют HTTP-сервера и backend. Для полноценной проверки используйте `pnpm dev` или `pnpm local`.
