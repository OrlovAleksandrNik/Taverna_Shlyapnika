# Troubleshooting

## Бот не отвечает

Проверьте:

- заполнен ли `TELEGRAM_BOT_TOKEN`;
- нет ли активного webhook при `BOT_MODE=polling`;
- не запущен ли второй экземпляр бота с тем же токеном;
- нет ли ошибки `409 Conflict`;
- доступен ли Telegram API из окружения;
- подключается ли база данных;
- что основной процесс действительно запускает `startBot()`.

## Афиша пустая

Проверьте:

- игра имеет статус `published`;
- дата игры в будущем;
- backend и сайт подключены к одной базе;
- `GET /api/games` возвращает игру;
- в браузере нет ошибки CORS;
- сайт открыт через HTTP, а не только через `file://`, если нужна динамика API.

## Заявка не отправляется

Проверьте:

- заполнены обязательные поля;
- установлен чекбокс согласия;
- frontend отправляет `consentGiven: true`, `consentVersion` и `privacyPolicyVersion`;
- backend не возвращает `CONSENT_REQUIRED`;
- база доступна.

## Prisma не подключается к базе

Проверьте:

- `DATABASE_URL`;
- запущен ли PostgreSQL;
- применены ли миграции;
- не занят ли порт `5432`;
- соответствует ли схема `public`.

## Кириллица отображается неправильно

Проверьте:

- `<meta charset="UTF-8">` на HTML-страницах;
- сохранение HTML/CSS/JS/TS в UTF-8;
- поддержку кириллицы у подключённых шрифтов;
- что текст не был случайно перекодирован в Windows-1251.

## Docker build падает

Проверьте:

- используется ли pnpm-lock.yaml;
- доступен ли registry npm;
- не попали ли `.env`, uploads, база или логи в build context;
- выполняется ли `pnpm prisma:generate` до `pnpm build`.
