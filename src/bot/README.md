# Telegram bot module

Этот модуль содержит Telegram-бота на grammY.

Главный файл: `index.ts`.

Бот использует:

- Prisma для мастеров, игр, сессий и рейтинга
- `status.ts` для health-check
- сервисы из `../services`
- utilities из `../utils`

При расширении бота держите состояние диалогов в базе или в отдельном сервисе, а не в глобальных переменных процесса.
